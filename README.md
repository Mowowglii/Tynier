# Tynier

Implémentation personnelle d'un compresseur utilisant l'algorithme **LZSS** en Rust.

## Build

```bash
cargo build --release
```
## Utilisation

./target/release/tynier --help

## État du projet

⚠️ Work in Progress

Apprentissages :

- Implémentation de l'algorithme LZSS

- Manipulation de structures de données optimisées en Rust

- Gestion des bits et sérialisation binair
